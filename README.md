# FilmPremieres

Android-приложение для просмотра информации о фильмах и премьерах.

## Features
- Список фильмов и премьер
- Поиск фильмов
- Экран деталей фильма
- Добавление фильмов в избранное
- Работа с сетью и локальным хранилищем

## Stack
- Kotlin
- Android SDK
- ViewBinding
- MVVM
- Coroutines
- Retrofit
- Room
- Dagger 2

## Architecture
- Clean Architecture (data / domain / presentation)
- MVVM
- Repository pattern
- UseCases
